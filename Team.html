Team Analysis

This is a representation of an entire team's (the Colorado Rockies) pitching staff's arm angles. This plot was built using the statcast data that MLB puts out daily. The pitchers mound would be placed at the '0' mark on the X axis and the Y axis is feet off the ground. so any dots to the left of 0 on the X axis is from a Right handed pitcher, and all those on the right side of 0 are lefty's (it is the view the batter and catcher have)

This is a plot that represents the same staff's (Rockies) pitch movement in the zone. Again this angle is the Catcher's perspective with 'pfx_x' on the x axis showing horizontal movement, and 'pfx_z' showing the vertical movement by pitch. The pitch type is the color scale used with legend to plot on side

This graph is showing pitches horizontal movement (x-axis) and vertical movement (y-axis)

While this image shows an inverted view of where each event landed and is color coded by the batted ball type
